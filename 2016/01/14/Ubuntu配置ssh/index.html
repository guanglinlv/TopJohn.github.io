Ubuntu配置ssh

By John

发表于 2016-01-14

文章目录

每次ssh连服务器都要敲一大段密码，不方便，之前做项目的时候都是给boss公钥，直接无密登录的，如今自己搞了一个服务器玩玩，想着也配一下吧。

  1. 先用终端ssh，用密码登录远程Ubuntu，输入sudo apt-get install openssh-server，下载安装openssh-server。

  2. 下载完成后系统会自动安装，都完成后看看sshserver启动没有。输入ps -e |grep ssh，如果有sshd那行就说明启动成功了。如果没有，就sudo /etc/init.d/ssh start，将其开启。

  3. 在自己的电脑上ssh-keygen -t rsa -C joyqi -f my-key-file，生成公钥和私钥。

  4. 将你的公钥放到服务器的~/.ssh/authorized_keys中就可以无密登录了。我用的Mac，

所以方便了很多。其它系统的可以参照http://segmentfault.com/a/1190000000481249，都不复杂。
注意如果你要改公钥的名字，那么连私钥的名字也改了，不然会出错，亲测！！！
