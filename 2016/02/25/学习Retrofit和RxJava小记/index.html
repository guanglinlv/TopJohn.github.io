学习Retrofit和RxJava小记

发表于 2016-02-25   |   分类于 Android   |  

Retrofit

  1. Retrofit官网学习：http://square.github.io/retrofit/

  2. 简书上的Retrofit 1.x 版本学习的不错博文，介绍了旧版本中的使用方法，不过旧版本的缺点就是不能cancel：http://www.jianshu.com/p/63723f593695

  3. 简书上的Retrofit 2.0.x 版本学习的博文，新版本支持cancel：http://www.jianshu.com/p/90b1f20b123d

  4. RxJava+Retrofit框架Demo，介绍了如何将Retrofit和RxJava进行封装：http://www.jianshu.com/p/2b0aeb6b6b61

RxJava

  1. 给Android开发者的RxJava详解：https://gank.io/post/560e15be2dca930e00da1083#toc_26

  2. 谜之RxJava （一） —— 最基本的观察者模式：https://segmentfault.com/a/1190000004049490

  3. 谜之RxJava （二） —— Magic Lift：https://segmentfault.com/a/1190000004049841

  4. 迷之RxJava （三）—— 线程切换：https://segmentfault.com/a/1190000004051191

  5. 迷之RxJava（四）—— Retrofit和RxJava的基情：https://segmentfault.com/a/1190000004077117

总结

Retrofit已经给了RxJava的实现，所以2者结合使用会更好，看了2天，虽然看懂了，但是实际运用还是有点欠缺，在项目中感悟下，下次再写篇实战吧。
